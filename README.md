# FaroTrackerAndroidApp

You must enter in the cards as a number for the number cards and as a captalized word for the courts so for instance: 
King of Hearts
King of Spades 
5 of Diamonds
2 of Hearts 

The suites must be capitalized. This is a work in progress.
